不管是移动端的浏览器还是 PC 端的浏览器,在宽度的表现上是一致的.为了将宽度说明白,需要先介绍三个概念. 物理像素, 设备像素, CSS 像素. 虽然然都是像素,但是所指不同.只有物理像素是客观的,无法通过程序手动修改(理所当然的),是由显卡的性能决定的.而设备像素和 CSS 像素都是人为了某种特定的目的而规定的.window 系统的电脑,控制面板,大家很熟悉的屏幕分辨率的设置,这里的分辨率就是设备像素.而 CSS 像素主要是为了布局而设置的,和 rem em 本质上是一种相对长度单位.

上面提到 CSS 像素是一个相对长度单位,那他的参照物是什么呢?或者说他的参照物的选取要遵循什么样的原则呢?只有一个原则,哪种选择对不同尺寸的电脑或者移动端更适配,更友好,获得的体验更一致.
第一个选择,物理像素,首先落选,因为物理像素无法更改,这会导致浏览器环境最大的复杂性.
第二个选择 设备像素, 设备像素和屏幕尺寸相关.一般屏幕尺寸越大其默认的设备像素越大;
在没有缩放的时候,1 CSS 像素和 1 设备像素
